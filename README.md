# 🏦 Simulatore di Mutui

Applicazione web interattiva per confrontare diversi scenari di mutuo, calcolare rate mensili e visualizzare graficamente la composizione dei pagamenti nel tempo.

## 🚀 Provalo subito

**👉 [Apri il Simulatore](https://pierpaolopalmiotti.github.io/simulatore-mutui/)**

## Funzionalità

- **Confronto multiplo**: aggiungi quanti mutui vuoi con capitale, tasso e durata personalizzabili
- **Calcolo automatico**: rata mensile, totale pagato e totale interessi con ammortamento alla francese
- **Grafici interattivi**:
  - Confronto rata mensile vs totale pagato
  - Composizione pagamenti nel tempo (capitale vs interessi)
  - Andamento del debito residuo
- **Preset di esempio**: configurazione rapida con 3 mutui preimpostati (20/25/30 anni)

## Tecnologie

- HTML / CSS / JavaScript
- [Chart.js](https://www.chartjs.org/) per i grafici interattivi
- Nessun backend richiesto — funziona interamente lato client

## Utilizzo in locale

Clona il repository e apri `index.html` nel browser:

```bash
git clone https://github.com/pierpaolopalmiotti/simulatore-mutui.git
cd simulatore-mutui
open index.html
```
